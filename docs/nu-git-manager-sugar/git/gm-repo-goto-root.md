# `gm repo goto root` (`nu-git-manager-sugar git`)
go to the root of the repository from anywhere in the worktree

## Examples
```nushell
# go back to the root of a repo
cd foo/bar/baz; gm repo goto root; print (pwd)
```
```
/path/to/repo
```
